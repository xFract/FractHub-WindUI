WindUI 使える要素一覧

基本
local Window = WindUI:CreateWindow({
	Title = "My UI",
	Theme = "Dark",
	Folder = "MyScript",
})

local MainSection = Window:Section({ Title = "Main Section" })
local MainTab = MainSection:Tab({ Title = "Main", Icon = "home" })

Window で使うもの
Window:Section({ Title = "Section Name" })
サイドバー内のグループを作る。

Window:Tab({ Title = "Main", Icon = "home" })
サイドバーに直接タブを作る。

Window:Tab({ Title = "Layout", Icon = "boxes", Columns = 2 })
`Box = true` の Section をタブ内で2カラム配置できる。
必要なら `MinColumnWidth = 180` などを付けて、何px相当で1カラムへ戻すか調整できる。

Section:Tab({ Title = "Main", Icon = "home" })
Section の中にタブを作る。

Window:SelectTab(1)
番号またはタブを指定して選択する。

MainTab:Select()
そのタブを開く。

Window:AddSettings()
設定タブを自動追加する。

Window:SetSidebarLogo("rbxassetid://...")
サイドバー上部ロゴを変更する。

Window:EditOpenButton({ BackgroundTransparency = 1 })
最小化後のボタン見た目を変更する。

タブ内で使える要素

Button
MainTab:Button({
	Title = "Notify",
	Callback = function()
		print("clicked")
	end,
})

Toggle
MainTab:Toggle({
	Title = "Auto Farm",
	Value = false,
	Callback = function(state)
		print(state)
	end,
})

Slider
MainTab:Slider({
	Title = "Distance",
	Step = 1,
	Value = { Min = 0, Max = 50, Default = 15 },
	Callback = function(value)
		print(value)
	end,
})

Input
MainTab:Input({
	Title = "Player Name",
	Placeholder = "Type here...",
	Callback = function(value)
		print(value)
	end,
})

Dropdown
MainTab:Dropdown({
	Title = "Position",
	Value = "Above",
	Values = { "Above", "Below", "Behind" },
	Callback = function(value)
		print(value)
	end,
})
単一選択。Refresh / Select も使える。

Multi Dropdown
MainTab:Dropdown({
	Title = "Select Items",
	Value = {},
	Values = { "Sword", "Shield", "Potion" },
	Multi = true,
	SearchBarEnabled = true,
	Callback = function(values)
		for name, selected in pairs(values) do
			print(name, selected)
		end
	end,
})
複数選択。Multi = true と Value = {} を使う。

Keybind
MainTab:Keybind({
	Title = "Toggle Key",
	Value = "RightShift",
	Callback = function(key)
		print(key)
	end,
})

Colorpicker
MainTab:Colorpicker({
	Title = "Theme Color",
	Default = Color3.fromRGB(0, 170, 255),
	Callback = function(color)
		print(color)
	end,
})

Paragraph
MainTab:Paragraph({
	Title = "Info",
	Desc = "説明文を置ける",
	Image = "info",
})

Divider
MainTab:Divider()

Space
MainTab:Space()

Image
MainTab:Image({
	Title = "Preview",
	Image = "rbxassetid://123456",
})

Code
MainTab:Code({
	Title = "Example",
	Code = "print('hello')",
})

Group
local Group = MainTab:Group()
Group:Button({ Title = "Test", Callback = function() end })
Group:Toggle({ Title = "Enabled", Value = true, Callback = function(v) end })
横並び用。中に Button / Toggle などを置ける。

例: Toggle を横並び
local FarmToggleGroup = FarmSection:Group()
FarmToggleGroup:Toggle({ Title = "Auto Farm", Flag = "auto_farm", Value = false, Callback = function(v) end })
FarmToggleGroup:Toggle({ Title = "Wait In Air", Flag = "hover_when_idle", Value = false, Callback = function(v) end })

Section
local FarmSection = MainTab:Section({
	Title = "Farm Settings",
	Desc = "Section inside tab",
	Box = true,
	BoxBorder = true,
	Opened = true,
	Columns = 2,
	MinColumnWidth = 140,
})

FarmSection:Toggle({ Title = "Auto Farm", Value = false, Callback = function(v) end })
FarmSection:Dropdown({ Title = "Position", Value = "Above", Values = { "Above", "Below" }, Callback = function(v) end })
FarmSection:Slider({ Title = "Distance", Step = 1, Value = { Min = 0, Max = 50, Default = 15 }, Callback = function(v) end })
タブ内の枠付きセクション。`Columns = 2` を指定すると中身を2カラムにできる。

例: Box Section を2カラム配置
local LayoutTab = MainSection:Tab({
	Title = "Layout",
	Icon = "boxes",
	Columns = 2,
	MinColumnWidth = 180,
})

local BoxSectionA = LayoutTab:Section({
	Title = "Section A",
	Box = true,
	BoxBorder = true,
	Opened = true,
})

local BoxSectionB = LayoutTab:Section({
	Title = "Section B",
	Box = true,
	BoxBorder = true,
	Opened = true,
})

BoxSectionA:Toggle({ Title = "Auto Farm", Value = false, Callback = function(v) end })
BoxSectionB:Dropdown({ Title = "Position", Value = "Above", Values = { "Above", "Below" }, Callback = function(v) end })

コンフィグ保存
対応要素は Toggle / Slider / Input / Dropdown / Keybind / Colorpicker。
保存したい要素に Flag を付ける。

MainTab:Toggle({
	Title = "Auto Farm",
	Flag = "auto_farm",
	Value = false,
	Callback = function(v) end,
})

MainTab:Dropdown({
	Title = "Position",
	Flag = "farm_position",
	Value = "Above",
	Values = { "Above", "Below", "Behind" },
	Callback = function(v) end,
})

local Config = Window.ConfigManager:Config("default", true)
Config:Save()
Config:Load()
Config:SetAutoLoad(true)

カスタム値も保存できる。
Config:Set("selected_world", "World1")
local world = Config:Get("selected_world")

補足
Folder を指定しないとコンフィグ保存は使えない。
保存先は WindUI/<Folder>/config/*.json。
Studio ではコンフィグ保存は動かない。

ロック対応
Button / Toggle / Slider / Input / Dropdown / Keybind / Colorpicker は Locked と LockedTitle を使える。
作成後は :Lock() / :Unlock() も使える。
